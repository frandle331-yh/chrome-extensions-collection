# Chrome Extensions Collection

A curated collection of productivity-focused Chrome extensions for developers, designers, and power users. Built with Manifest V3 and vanilla JavaScript for maximum performance and minimal footprint.

Available on the [Chrome Web Store](https://chromewebstore.google.com/).

---

## Extensions

### Free Extensions

| Extension | Description |
|-----------|-------------|
| **MarkdownCopy** | Copy web page text as clean Markdown. Supports headings, bold/italic, links, lists, code blocks, tables, images, and blockquotes. |
| **PageMeta** | Inspect page metadata in one click — OGP tags, Twitter Cards, heading structure, and all meta tags with copy support. |
| **SelectTools** | Right-click text utilities: case conversion, Base64/URL encode-decode, hash generation, JSON format, word count, and more. |
| **JSONView** | Auto-format JSON responses in the browser with syntax highlighting, collapsible tree view, search, and one-click copy. |
| **QuickNote** | Attach notes to any URL. Auto-saves locally, searchable note list, Markdown export. |
| **RegexPlayground** | Live regex tester with real-time matching, capture group display, flag controls, and a built-in cheat sheet. |
| **ColorPeek** | Extract the full color palette from any web page. View in HEX/RGB/HSL, sorted by frequency, copy as CSS variables. |
| **ResponsePeek** | View HTTP response headers with security header scoring, cache analysis, and full header list. |

### Premium Extensions

| Extension | Description |
|-----------|-------------|
| **EnvSwitch** | Switch between dev, staging, and production environments instantly. URL rewriting and header injection in one click. |
| **SnapDiff** | Capture page screenshots and compare them to spot visual changes. Pixel-level diff with highlighting. No cloud needed. |

## Installation

### From Chrome Web Store

Visit each extension's page on the [Chrome Web Store](https://chromewebstore.google.com/) and click "Add to Chrome".

### For Development

1. Open `chrome://extensions` in Chrome
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked** and select an extension folder

## Tech Stack

- **Manifest V3** — latest Chrome extension standard
- **Vanilla JavaScript** — no frameworks, lightweight and fast
- **Chrome APIs** — contextMenus, activeTab, scripting, tabs, storage, declarativeNetRequest
