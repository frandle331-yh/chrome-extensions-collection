# DevToolBox

無料のオンライン開発者ツール集。すべてブラウザ上で動作し、データが外部に送信されることは一切ありません。

## ツール一覧

| ツール | 説明 |
|--------|------|
| **JSON整形・検証** | JSONデータの整形、圧縮、バリデーション |
| **Base64 エンコード/デコード** | テキストとBase64の相互変換（UTF-8対応） |
| **パスワード生成** | 暗号学的に安全なランダムパスワードを生成 |
| **QRコード作成** | テキスト・URLからQRコードを生成、PNGダウンロード |
| **カラー変換** | HEX・RGB・HSLの相互変換 |
| **ハッシュ生成** | SHA-1、SHA-256、SHA-384、SHA-512ハッシュ値を一括生成 |
| **URLエンコード/デコード** | URLコンポーネントやURIのエンコード・デコード |

## 技術スタック

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **TypeScript**

## 開発の始め方

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## デプロイ方法

### Vercel（推奨・無料）

1. GitHubにリポジトリを作成してプッシュ
2. [vercel.com](https://vercel.com) にGitHubアカウントでログイン
3. リポジトリをインポート
4. 「Deploy」をクリック

Vercelの無料枠に含まれるもの：
- 無制限のデプロイ
- 自動HTTPS
- カスタムドメイン対応
- エッジネットワーク（CDN）

### 環境変数

| 変数名 | 説明 |
|--------|------|
| `NEXT_PUBLIC_SITE_URL` | サイトのURL（サイトマップ・SEO用）。例: `https://devtoolbox.example.com` |

## 収益化（Google AdSense）

アプリにはGoogle AdSense用の広告スロットがあらかじめ設置されています：

1. [Google AdSense](https://www.google.com/adsense/) に登録
2. 広告ユニットコードを取得
3. 広告プレースホルダーをAdSenseコードに差し替え
4. 広告スロットの配置場所：
   - ページ下部のバナー（全ページ共通・layout.tsx）
   - ツールページ右側のサイドバー（ToolLayout.tsx）

### 想定収益

SEOからの安定的なトラフィックがある場合：
- 日間500〜1000アクセス: 約$10〜30/月
- 日間1000〜3000アクセス: 約$30〜90/月
- 各ツールページが独立したSEOランディングページとして機能

## SEO対策

- 各ツールに最適化されたメタタイトル・ディスクリプション（日本語）
- サイトマップ（`/sitemap.xml`）自動生成
- `robots.txt` 自動生成
- 全ツールがクライアントサイド処理（高速読み込み = SEOに有利）
- ツールを追加するほど検索カバレッジが拡大

## 新しいツールの追加方法

1. `src/app/ツール名/` ディレクトリを作成
2. `page.tsx`（ツールのUI）と `layout.tsx`（SEOメタデータ）を追加
3. `ToolLayout` コンポーネントで統一的なレイアウトを適用
4. `src/app/page.tsx` のツールグリッドに追加
5. `src/app/layout.tsx` のナビゲーションに追加
6. サイトマップは自動的に更新されます
