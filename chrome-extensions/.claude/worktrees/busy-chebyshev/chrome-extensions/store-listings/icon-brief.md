# Chrome拡張機能 アイコン生成ブリーフ

## 共通要件

- サイズ: 128x128px（Chrome Web Store用。後で48x48, 16x16にリサイズ）
- 形式: PNG（透過背景推奨）
- スタイル: フラットデザイン / シンプル / モダン
- 背景: 角丸の正方形 or 円形の背景色 + 白アイコン（統一感を出す場合）
- 用途: Chrome拡張機能のアイコン（小さく表示されても識別できるシンプルさが必要）
- 細かいテキストは入れない（16pxで潰れるため）

---

## 1. Color Peek

- **機能**: Webページ内の全カラーを抽出し、HEX/RGB/HSLで表示。ワンクリックコピー
- **イメージ**: スポイト（カラーピッカー）/ カラーパレット / 虹色のドロップ
- **カラー**: 虹色グラデーション or カラフルな配色
- **キーワード**: 色抽出、スポイト、パレット

## 2. JSON View

- **機能**: ブラウザ上でJSONを自動整形。ツリービュー、シンタックスハイライト、検索
- **イメージ**: 波括弧 `{ }` / ツリー構造の図 / コードブロック
- **カラー**: ダークブルー or グリーン系（コード・開発者ツール感）
- **キーワード**: JSON、構造化データ、フォーマット

## 3. Markdown Copy

- **機能**: 選択したWebコンテンツをMarkdown形式でコピー。Obsidian/Notion/GitHub向け
- **イメージ**: Markdownの「M↓」ロゴ風 / コピーアイコン+Md記号 / クリップボード+マークダウン記号
- **カラー**: 黒 or ダークグレー背景に白（Markdownのシンプルさ）
- **キーワード**: Markdown、コピー、テキスト変換

## 4. Page Meta

- **機能**: OGPタグ、メタ情報、見出し構造を一覧表示。SEO・Web開発向け
- **イメージ**: `</>` タグアイコン / 虫眼鏡+Webページ / 情報カード「i」マーク
- **カラー**: オレンジ or ブルー（情報・分析ツール感）
- **キーワード**: メタタグ、SEO、ページ情報、HTMLタグ

## 5. Quick Note

- **機能**: Webページに紐づくメモを即座に作成・自動保存・再訪問時に復元
- **イメージ**: 付箋（ポストイット）/ メモ帳+ペン / 吹き出しメモ
- **カラー**: 黄色 or アンバー（付箋らしさ）
- **キーワード**: メモ、付箋、ノート、書き込み

## 6. Regex Playground

- **機能**: 正規表現をリアルタイムテスト。ライブマッチング、グループハイライト、チートシート
- **イメージ**: `.*` や `/regex/` のような正規表現記号 / ターミナル風 / 検索パターン
- **カラー**: パープル or グリーン（ターミナル・コード感）
- **キーワード**: 正規表現、パターンマッチング、テスト

## 7. Response Peek

- **機能**: HTTPレスポンスヘッダー、セキュリティヘッダー、キャッシュ情報を即座に確認
- **イメージ**: ネットワーク通信アイコン（↕矢印）/ HTTP文字+虫眼鏡 / サーバーレスポンスの稲妻
- **カラー**: レッド or ダークブルー（ネットワーク・セキュリティ感）
- **キーワード**: HTTP、レスポンス、ヘッダー、ネットワーク

## 8. Select Tools

- **機能**: 選択テキストを右クリックで文字数カウント、大小変換、Base64、URLエンコード、ハッシュ生成
- **イメージ**: テキスト選択カーソル（I-beam）+工具 / スイスアーミーナイフ / ツールボックス+テキスト「Aa」
- **カラー**: ブルー or ティール（ユーティリティ・万能ツール感）
- **キーワード**: テキスト変換、マルチツール、ユーティリティ
