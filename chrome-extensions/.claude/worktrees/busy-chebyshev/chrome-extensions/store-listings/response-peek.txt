Response Peek - HTTP Response Header Viewer

Instantly view HTTP response headers, security headers, caching info, and server details for any page.

Features:
• View all response headers at a glance
• Check security headers (CSP, HSTS, X-Frame-Options, etc.)
• Inspect caching settings (Cache-Control, ETag, etc.)
• Display server information

Perfect for:
• Backend developers — verify and debug header configurations
• Security engineers — audit security headers
• Infrastructure engineers — validate caching settings
• Web developers — inspect production response details
