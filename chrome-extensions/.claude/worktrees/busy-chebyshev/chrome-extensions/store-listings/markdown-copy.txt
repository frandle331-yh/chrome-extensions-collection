Markdown Copy - Copy as Markdown

Copy selected webpage content as clean Markdown. Built for Obsidian, Notion, and GitHub users.

Features:
• Right-click to copy selection as Markdown
• Accurately converts headings, lists, links, and images
• Preserves tables in Markdown format
• Supports code blocks

Perfect for:
• Obsidian users — quickly import web content into your notes
• Notion users — paste clean formatted content
• GitHub users — gather material for issues and READMEs
• Bloggers — clip reference material effortlessly
