Select Tools - Text Utility Toolkit

Right-click selected text to instantly count characters, convert case, encode/decode Base64 & URL, generate hashes, and more.

Features:
• Character and word count
• Case conversion (UPPER / lower / Title Case)
• Base64 encode / decode
• URL encode / decode
• Hash generation (MD5, SHA-1, SHA-256)
• Instant access from the right-click context menu

Perfect for:
• Developers — encode, decode, and hash without leaving the browser
• Writers — quick character counts
• Marketers — everyday text transformation tool
